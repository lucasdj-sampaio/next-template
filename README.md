##  Getting Started

1. Clone the repository:
    ```bash
    git clone https://github.com/lucasdj-sampaio/next-template.git
    cd next-template
    ```

2. Install dependencies:
    ```bash
    npm install
    # or
    yarn install
    ```

3. Run your development server:
    ```bash
    npm run dev
    # or
    yarn dev
    ```
   Open [http://localhost:3000](http://localhost:3000) in your browser to see it live.

4. Start building by editing `app/page.tsx` (or `pages/index.tsx` depending on your structure) ‚Äî your changes will reflect automatically.
